# Terminal-Calculator
Un script simple, y extendible para terminales. El programa esta en beta lo cual tiene muchos bugs.

## Instalacion
```bash
python3 setup.py
```

Intalara el modulo de colorana, para que muestre los colores por salida en la terminal.


# Ejecutar el Script
```bash
cd src/
python3 main.py
```
